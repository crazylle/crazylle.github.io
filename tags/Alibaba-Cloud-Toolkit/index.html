IDEA http client无法解析enviroment file

听说IDEA的http client功能挺好用的，就想把之前用Postman测试的接口转换到IDEA中，结果前后尝试了好几次都没尝试成功，这两天终于狠下心找到了原因。

问题描述

创建了环境变量文件http-client.env.json，然后在*.http文件中引用环境变量，运行HTTP请求无法读取环境变量文件中定义的变量。
